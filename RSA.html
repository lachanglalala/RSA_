RSA

The basic purpose of RSA is to reveal differences among genes and transcripts from different states of the same species using a simple approach that does not require familiarity with Linux and algorithm implementation.

Quick Start

Prerequisites

  • Linux OS (required)

Download

You can download and install the RSA in  RSA

Flow Chart

All steps from raw read quality control to finding out differences among genes and transcripts.

Configuration

RSA will install all the software and configure the PATH. We unified installed in /opt/biosoft/ folder below. This step include set up PATH, but you can set install folder and configuration PATH according to your habits. Install FastQC in /opt/biosoft/ or you can modify the running script

Run

First Running RSAprepare 

$sh RSAprepare

It will make a “welcome” work folder, which contains two folders named “control” and “experiment”. Put two groups FASTQ data which need to be analyze in “control” folder and “experiment” folder respectively. Put genome sequence files and genome annotation file in “welcome” folder.

Make sure put all the files in respectively folders. If your data is single-ended sequencing choose RSA_single_lncRNA or RSA_single_mRNA. If your data is pair-ended sequencing choose RSA_pair_end_lncRNA or RSA_pair_end_mRNA.

$sh RSA_single_lncRNA

$sh RSA_single_mRNA

$sh RSA_pair_end_lncRNA

$sh RSA_pair_end_mRNA

FAQs

  1. Why run fail?

    The software version of RSA software folder may different with you download. When the installation of the PATH variable is not set successfully you can modify the running script inside the call PATH address resolved.

  2. How long does it take to run?

    It depends on your computer configuration and the amount of data.

  3. How high is the computer configuration?

    Recommended hardware configuration (64-bit computer running either Linux or Mac OS X (10.4 Tiger or later); 4 GB of RAM (16 GB preferred).

  4. What is the advantage of this software?

    RSA can substantially lower the effort required for scientists entering into the areas of RNA-Seq and Big Data, especially for those not familiar with Linux.
